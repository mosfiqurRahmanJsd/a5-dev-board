Question-1: What are the different ways to select an element in the DOM?

Answer: 1. document.getElementById(), 2. document.getElementsByClassName(), 3. document.querySelector(), 4. document.querySelectorAll(), 5. document.getElementByTagName()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML is html element, innerText is html element content The textContent property also ignores all HTML tags and returns only the text.

Question-3: What is event delegation in the DOM?

Answer: By default, events triggered on an element propagate up the DOM tree to the element's parent, its ancestors, and on up until the root element (html).

Question-4: What is event bubbling in the DOM?

Answer: HTML elements receive different types of events, from click, to blur, to scroll, and so on.
